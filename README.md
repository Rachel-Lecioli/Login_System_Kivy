# Sistema de Login com Kivy

Aplicação de login e criação de contas desenvolvida em Python utilizando Kivy.

## Funcionalidades
- Cadastro de usuários
- Login
- Validação de conta
- Interface gráfica
- Armazenamento em arquivo `.txt`

## Tecnologias
- Python
- Kivy

## Estrutura
- main.py → lógica principal
- database.py → gerenciamento dos usuários
- my.kv → interface gráfica

## Como executar

Instale as dependências:

pip install -r requirements.txt

Depois execute:

python main.py
